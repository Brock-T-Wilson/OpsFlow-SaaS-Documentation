# Notification Actions

Notification actions automatically alert users when operational events occur.

## Notification Channels

OpsFlow supports:

- email notifications
- in-platform alerts
- SMS notifications
- Slack integrations
- Microsoft Teams integrations

## Notification Events

Notifications can be triggered by:

- incident submissions
- workflow approvals
- scheduling changes
- escalations
- overdue tasks

## Best Practices

- Avoid excessive notification frequency.
- Configure escalation priorities carefully.
- Review notification delivery logs regularly.
