# Escalation Rules

Escalation rules automatically route unresolved operational issues
to additional personnel or management levels.

## Configure Escalation Rules

1. Open **Automation > Escalation Rules**.
2. Select **Create Rule**.
3. Define escalation trigger conditions.
4. Configure escalation recipients.
5. Set escalation timing thresholds.
6. Save changes.

## Example Escalation Triggers

| Trigger | Escalation Action |
|---|---|
| Critical incident | Notify supervisors immediately |
| Overdue work order | Escalate to department manager |
| Staffing shortage | Notify operations leadership |
