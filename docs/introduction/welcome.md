# Welcome to OpsFlow

OpsFlow is an enterprise workforce operations platform
designed to help organizations manage operational workflows,
incident reporting, scheduling, automation, and reporting
from a centralized system.

## Key Capabilities

- Workforce scheduling
- Incident management
- Workflow automation
- Reporting and analytics
- Role-based permissions
- Security administration
- API integrations

## Intended Audience

This documentation is intended for:

- Employees
- Supervisors
- Administrators
- Operations managers
- IT teams
- Developers integrating with the OpsFlow API
