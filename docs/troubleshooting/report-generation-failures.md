# Report Generation Failures

Report generation issues may occur when exporting large operational datasets.

## Common Causes

Reports may fail because of:

- excessive data volume
- expired export sessions
- insufficient permissions
- timeout restrictions

## Resolution

1. Reduce report date range.
2. Retry the export request.
3. Verify export permissions.
4. Generate reports during lower traffic periods.

## Supported Export Formats

OpsFlow supports:

- CSV exports
- PDF exports
- XLSX exports
