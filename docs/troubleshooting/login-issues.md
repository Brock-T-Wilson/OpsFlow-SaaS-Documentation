# Login Issues

This guide explains how to troubleshoot common OpsFlow login problems.

## Invalid Username or Password

Verify that:

- the correct email address is used
- the password meets current security requirements
- Caps Lock is disabled

## MFA Verification Failure

If MFA verification fails:

1. Verify automatic device time synchronization is enabled.
2. Retry the verification code.
3. Use a backup recovery code if available.

## Account Locked

Accounts may become temporarily locked after multiple failed login attempts.

Contact your administrator if access cannot be restored.
