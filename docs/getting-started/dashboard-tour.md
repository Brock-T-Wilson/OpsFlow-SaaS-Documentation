# Dashboard Tour

The OpsFlow dashboard provides centralized access to operational activity,
reporting, scheduling, and workflow notifications.

## Dashboard Sections

### KPI Overview

Displays:
- open incidents
- overdue tasks
- active work orders
- staffing levels

### Notifications Panel

Displays:
- escalation alerts
- approval requests
- workflow failures
- scheduling updates

### Analytics Widgets

Provides visibility into:
- productivity trends
- incident frequency
- operational performance

## Customize Dashboard Layout

Users can:

1. Select **Edit Dashboard**.
2. Rearrange dashboard widgets.
3. Save custom layouts.
