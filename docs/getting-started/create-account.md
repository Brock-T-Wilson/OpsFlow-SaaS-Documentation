# Create an Account

This guide explains how to create a new OpsFlow account.

## Before You Begin

You will need:

- A valid company email address
- An invitation link from your organization administrator
- Access to multi-factor authentication (MFA)

## Create Your Account

1. Open the invitation email from OpsFlow.
2. Select **Activate Account**.
3. Enter your first and last name.
4. Create a password that meets security requirements.
5. Configure MFA using an authenticator application.
6. Select **Complete Registration**.

## Password Requirements

Passwords must contain:

- Minimum 12 characters
- One uppercase letter
- One lowercase letter
- One number
- One special character

## Expected Result

After registration is complete, you will be redirected to the OpsFlow dashboard.

## Troubleshooting

### Invalid Invitation Link

Invitation links expire after 72 hours.

Contact your administrator to request a new invitation.

### MFA Code Rejected

Verify that your device time is synchronized automatically before retrying.
