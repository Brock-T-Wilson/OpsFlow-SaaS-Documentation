# Organization Setup

Organization setup allows administrators to configure departments,
locations, and operational teams within OpsFlow.

## Before You Begin

You must have:

- Administrator permissions
- An active organization account

## Configure Departments

1. Open **Administration > Departments**.
2. Select **Create Department**.
3. Enter a department name.
4. Assign department managers.
5. Select **Save Changes**.

## Recommended Department Structure

Typical departments include:

- Operations
- Maintenance
- Logistics
- Safety
- Human Resources

## Expected Result

Departments become available for:
- scheduling
- work order assignment
- reporting
- incident tracking
