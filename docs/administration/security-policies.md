# Security Policies

OpsFlow security policies help organizations protect operational data,
user accounts, and administrative workflows.

## Password Policy Requirements

Passwords must contain:

- minimum 12 characters
- uppercase and lowercase letters
- numeric characters
- special characters

## Session Timeout Policy

Inactive sessions automatically expire after:

| Role | Timeout |
|---|---|
| Employee | 30 minutes |
| Supervisor | 20 minutes |
| Administrator | 15 minutes |

## Administrative Recommendations

- Enable MFA for all privileged users.
- Review audit logs weekly.
- Disable inactive accounts promptly.
- Restrict unnecessary administrative access.
