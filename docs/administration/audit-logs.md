# Audit Logs

Audit logs provide visibility into user activity,
security events, and administrative actions.

## Logged Activities

OpsFlow records:

- login attempts
- permission changes
- workflow modifications
- incident escalations
- API authentication events
- report exports

## Review Audit Logs

1. Open **Administration > Audit Logs**.
2. Apply filters by user, event type, or date.
3. Review activity records.

## Audit Log Retention

| Log Type | Retention Period |
|---|---|
| Security Events | 365 days |
| Workflow Events | 180 days |
| User Activity | 90 days |
