# API Questions

## How do I generate an API key?

1. Open **Administration > API Access**.
2. Select **Generate API Key**.
3. Configure API permissions.

---

## What authentication method does the API use?

OpsFlow APIs use bearer token authentication.

---

## Does the API support JSON?

Yes. All API requests and responses use JSON formatting.

---

## Are API requests rate limited?

Yes. API usage limits apply to maintain platform stability.
