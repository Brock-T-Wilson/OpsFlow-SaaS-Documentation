# Shift Scheduling

The Shift Scheduling module allows supervisors to assign staffing coverage,
manage overtime, and coordinate operational schedules.

## Create a Shift Schedule

1. Open **Scheduling > Shift Planner**.
2. Select a department.
3. Choose a scheduling period.
4. Assign employees to available shifts.
5. Review staffing coverage.
6. Select **Publish Schedule**.

## Schedule Status Types

| Status | Description |
|---|---|
| Draft | Schedule not yet published |
| Published | Visible to employees |
| Locked | Schedule cannot be modified |

## Best Practices

- Review overtime distribution weekly.
- Avoid overlapping assignments.
- Validate staffing minimums before publishing.
