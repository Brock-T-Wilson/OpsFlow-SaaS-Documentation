# Known Issues

This page tracks active platform issues and operational limitations.

## Dashboard Export Timeout

Large PDF exports may fail during peak operational periods.

### Workaround

Reduce export date ranges before retrying.

---

## Delayed SMS Notifications

SMS notifications may experience delayed delivery
in low-connectivity environments.

### Workaround

Enable email notifications as a secondary delivery method.

---

## API Rate Limit Visibility

Some API responses may not display retry timing headers consistently.

### Status

Under investigation.
